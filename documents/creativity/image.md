## CONCEPT 1: "The Lie Detector"

**Narrative:** Dashboard on trial — revealing that SaaS analytics are lying while Google tells the truth.

### AI Image Generation Prompt:

```
A dramatic courtroom scene photograph, ultra-realistic, cinematic composition. 
Central focus: A glowing laptop screen displaying a colorful fake analytics 
dashboard with inflated numbers, placed on a wooden witness stand. Next to it, 
a second laptop shows the real Google Search Console interface with lower, 
honest numbers. A golden gavel hovers mid-air, frozen in motion about to strike. 
Dark courtroom background with dramatic overhead spotlight illuminating only 
the two laptops. Volumetric light rays visible. Red digital text floating in 
the air: "LIES vs TRUTH". Shot on Hasselblad H6D-100c, 50mm lens, f/2.8, 
shallow depth of field, ISO 400. Color palette: deep navy blue, gold accents, 
red warning highlights, cream lighting. Style: David Fincher cinematography 
meets tech editorial photography. --ar 16:9 --q 2 --style raw --v 6

Negative prompt: cartoon, illustration, low quality, blurry, amateur, clipart, 
anime, 3D render looking fake, distorted text, multiple gavels, cluttered 
background, bright cheerful colors, childish elements
```

**Alternative Variation (Square for Instagram):**
```
Same concept, vertical composition, 1:1 aspect ratio, tighter framing on the 
two laptops with gavel, more intimate and bold --ar 1:1
```

## 📝 USAGE INSTRUCTIONS FOR NANO BANANA

### For Best Results:

1. **Copy the full prompt** (everything in the code blocks above)
2. **Paste directly into Nano Banana** without modifications first
3. **If result isn't perfect, iterate by:**
   - Adjusting color palette keywords
   - Changing camera/lens specifications
   - Modifying lighting description
   - Adding more style references
4. **Generate 4 variations** per prompt to choose best
5. **Upscale the winning image** for production use

### Text/Arabic Elements:
- AI image generators often struggle with accurate text
- Generate the visual first, then add Arabic text in Figma/Photoshop post-production
- If you need Arabic text in the image, use simple, short phrases only

### Aspect Ratio Guide:
- `--ar 16:9` → LinkedIn, YouTube thumbnails, website banners
- `--ar 1:1` → Instagram feed, LinkedIn posts
- `--ar 9:16` → Instagram Stories, TikTok, Reels, Snapchat
- `--ar 4:5` → Instagram feed (portrait)

### Quality Parameters:
- `--q 2` → Maximum quality
- `--style raw` → Less stylized, more photographic
- `--v 6` → Latest Midjourney version (or equivalent)

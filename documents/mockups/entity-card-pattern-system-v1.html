📋 قرارات مفتوحة

✅

الـ anatomy موحّد: accent strip + inset image + title + footer 70/30

✅

الأثر الرقمي = impressions (ظهور شهري) من GSC — رقم مباشر، بلا ألوان

❓

Tags — كروت كاملة أو تبقى pills؟

التوحيد يقترح كروت كاملة. لكن Tags كثيرة جداً عادةً (50-100 وسم) — الكروت الكاملة تثقّل الصفحة. توصيتي: نبقّي pills ولكن نطوّر شكلها (نضيف Avatar micro-stack صغير داخل الـ pill). قرار خالد.

❓

هل نبدأ بـ EntityCard implementation ولا نكمّل categories page أول؟

توصيتي: EntityCard مكوّن واحد مشترك أولاً → نطبّقه على categories (اللي معتمد) → ثم industries → ثم tags. الترتيب يضمن الـ system بدل الـ copy-paste.

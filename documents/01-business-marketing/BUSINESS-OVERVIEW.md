## 1. What Is Modonty

**Tagline**: "Presence, Not Promises"

Modonty is a subscription-based Arabic content platform. Businesses subscribe and receive professionally written, SEO-optimized Arabic articles published on Modonty's central Authority Blog — with strategic backlinks pointing to the client's website. Premium clients also get a customized version of each article for their own website.

**The core model in one sentence**: We write → we publish → your website gets backlinks → Google finds you.

## 5. The Four Powers (Customer-Facing)

### 1. Intelligent Lead Scoring
Algorithm assigns 0–100 score per visitor:
- 25% Frequency (return visits)
- 25% Depth (active reading time)
- 25% Interaction (scroll, clicks, shares)
- 25% Conversion (key events triggered)

Result: Filter leads by HOT (>70), WARM (>40), COLD — sales team knows who to call first.

### 2. SEO on Autopilot
Auto-generates JSON-LD, meta tags, Open Graph, Twitter Cards, canonical URLs. No manual SEO work needed.

### 3. Social Proof Ecosystem
Verified badge (Pro+), review aggregation from comments, follower system (builds re-targetable audience), Facepile display for trust signals.

### 4. Competitive Intelligence
Monitor competitor websites from the dashboard. Compare engagement scores. See what's working for competitors.

## 10. Dev Guidelines (for AI Agents)

**When adding features, ask**:
1. Does this strengthen the Authority Blog?
2. Does this improve SEO rankings?
3. Does this affect subscription value or tier differentiation?
4. Does this build client trust and transparency?

**Feature priority**:
- 🔴 High: SEO optimization, content quality, analytics, performance
- 🟡 Medium: Admin enhancements, content tools, client management
- 🟢 Low: UI polish, experimental features, non-critical integrations

**Business rules — must never break**:
- Authority Blog system and backlink value
- Subscription tier differentiation (Basic ≠ Premium)
- 18 months for 12 months payment promise
- GTM integration for transparency
- Premium-only: customized article version + CMS integration

**Database flow**:
```
Subscription → Monthly Quota → Content Creation →
Authority Blog Publishing → SEO Ranking →
Organic Traffic → Backlinks → Client Results → Renewal
```

**Key DB entities**: Client (tier, GTM ID, dates), Article (SEO fields, status, client link), Author (E-E-A-T signals), Analytics (engagement, traffic sources, Core Web Vitals)

**Platform mapping**:
- `admin/` — Internal dashboard (jbrtechno.com)
- `modonty/` — Authority Blog (Modonty.com)
- `console/` — Client-facing dashboard (jrbseo.com)
- `dataLayer/` — Shared Prisma database

# Category Detail Components

Route-specific components for the category detail page.

## Server Components (No JavaScript)
- `category-detail-hero.tsx` - Hero section with category image and stats
- `category-article-card.tsx` - Enhanced article card with optimized images
- `category-article-list-item.tsx` - List view alternative
- `article-skeleton.tsx` - Loading skeleton
- `category-empty-state.tsx` - No articles state
- `related-categories.tsx` - Related categories section

## Client Components (Minimal JavaScript)
- `article-search-form.tsx` - Search input with URL state
- `article-filters.tsx` - Sort and filter controls
